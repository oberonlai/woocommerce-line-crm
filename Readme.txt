== Description ==

OrderChatz 是一款專為 WooCommerce 打造的客服管理工具，可以直接在後台或網頁 App 與 LINE 好友即時進行聊天互動，並整合訂單資訊以及推播功能的實用外掛。

產品功能完整介紹：https://oberonlai.blog/order-chatz-marketing

產品說明文件：https://oberonlai.blog/docs/order-chatz-doc/

聊天介面功能：
-新加入官方帳號的好友會自動顯示於好友清單
-有新訊息的好友會自動顯示於好友清單
-好友名稱搜尋
-可拖曳圖片至聊天串進行上傳，所有聊天串中的圖片會存在 wp-content/uploads/order-chatz 裡面
-可傳送商品資訊，用商品名稱搜尋，不用切換頁面尋找
-可於右側欄綁定會員，綁定後就能顯示基本資訊與訂單紀錄
-可下標籤、下備註
-可搜尋訂單編號
-可查看訂單編輯頁，直接於同一畫面做訂單操作
-有新訊息進來時 admin menu 會顯示未讀訊息
-聊天介面支援手機版，並可接收新訊息推播

推播功能：
-可直接廣播所有好友
-可用條件篩選，目前支援根據標籤、會員綁定狀態、註冊日期以及最後訂單日期進行篩選
-可推播文字或是圖片
-可發送測試推播給 LINE Messaging Channel 管理員

好友管理功能
-可直接匯入帶有 LINE user ID 的網站會員，OrderNotify 限定
-可匯入官方帳號好友清單
-可以於好友列表頁設定會員綁定

標籤管理功能
-可查看標籤的好友清單

備註管理功能
-可查看與搜尋備註紀錄
-可用備註分類進行篩選

匯出功能
-可篩選日期會出聊天記錄
-可查看相關資料表的大小以便管理
-可下載聊天串中上傳的所有圖片

統計功能
-可查看訊息使用量
-當訊息使用量剩餘不到 10% 時會出現通知提醒
-可查看訊息類型使用比例
-可查看每日訊息類型使用量
